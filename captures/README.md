# captures
> 🌐 Read this in: **한국어** · [English](./README.en.md)

RS485 패킷 캡처와 분석 산출물을 임시로 보관하는 위치입니다. 기본적으로 바이너리 캡처와 원시 로그는 `.gitignore`로 제외합니다.

## 기록 권장 항목

| 항목 | 설명 |
|---|---|
| 날짜/시간 | 캡처 시각과 timezone |
| 보드 | AtomS3-Lite 또는 Atom-Lite |
| 펌웨어 버전 | `FW_VERSION` |
| UART 설정 | baud/data/stop/parity |
| 배선 | A/B/GND, 종단저항 상태 |
| 상황 | 부팅, 버튼, 월패드 조작 등 |

## 익명화

- 세대 식별값, 주소, 개인 식별 가능 문자열은 공유 전 제거합니다.
- 캡처 파일명에 집 주소나 실제 장비 위치를 넣지 않습니다.
- 공개 이슈에는 필요한 최소 HEX excerpt만 올립니다.
