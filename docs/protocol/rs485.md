## RX 수신

RX 바이트는 timestamp와 함께 수집하고, Web Console에는 Raw/HEX dump 형태로 노출합니다. Overflow는 UART overflow, queue overflow, dropped frame으로 분리해 Metrics에 기록합니다.

## TX 경로

1. 사용자가 HEX 문자열을 입력합니다.
2. 공백을 허용하되 2자리 HEX byte 단위로 검증합니다.
3. 검증된 byte 배열을 TX queue에 넣습니다.
4. RS485 드라이버가 DE/RE를 송신 모드로 전환합니다.
5. UART write와 flush 후 수신 모드로 복귀합니다.
