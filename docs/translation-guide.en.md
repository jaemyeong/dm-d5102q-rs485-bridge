# Translation Guide
> 🌐 Read this in: [한국어](./translation-guide.md) · **English**

## Policy

- Korean documentation is primary, and matching `.en.md` English documents are maintained together.
- Use the glossary below when a technical term can be translated multiple ways.
- Keep PRD section numbers identical across both languages.

## Writing Rules

- Keep heading structures aligned across Korean and English pairs.
- Mark undecided values with `TODO` and do not invent final decisions.
- For security-related text, describe access limits and intended scope before feature promotion.
