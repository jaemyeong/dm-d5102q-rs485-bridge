# 번역 가이드
> 🌐 Read this in: **한국어** · [English](./translation-guide.en.md)

## 정책

- 한국어 문서를 메인으로 작성하고 `.en.md` 영어 문서를 함께 유지합니다.
- 의미가 갈리는 기술 용어는 아래 용어표를 우선합니다.
- PRD 조항 번호는 양쪽 문서에서 동일하게 유지합니다.

## 용어 시드

| 한국어 | English |
|---|---|
| 월패드 | wall pad |
| 종단저항 | termination resistor |
| DE/RE | Driver Enable / Receiver Enable |
| 보레이트 | baud rate |
| 프레이밍 | framing |
| 캡처 | capture |
| 인젝션 | injection |
| 패시브 모니터링 | passive monitoring |
| 수동 명령 송신 | manual frame send |
| 자가 진단 | self-diagnostics |

## 문서 작성 규칙

- 한·영 페어의 제목 구조를 맞춥니다.
- 확정되지 않은 값은 `TODO`로 표시하고 임의로 확정하지 않습니다.
- 보안 관련 문구는 기능 홍보보다 접근 제한과 사용 범위를 먼저 설명합니다.
