# Web Admin Header Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the 46px single-row admin header with a hybrid persistent topbar + click-to-expand drawer that surfaces device health, alerts, and a reboot action; add the supporting `GET /api/info` and `POST /api/reboot` endpoints; bump firmware to 0.1.10.

**Architecture:** Master changes live in `shared/`; the existing `tools/sync_shared.sh` (which calls `tools/embed_web_ui.mjs`) propagates to `firmware/atom-lite/` and `firmware/atoms3-lite/` and regenerates the gzipped `admin_ui.h`. The HTML is one file (`shared/data/index.html`) containing markup + CSS + inline JS. Pure functions are isolated in a `<script id="header-pure-fns">` tag so unit tests can extract them via `vm.runInContext`.

**Tech Stack:**
- Arduino-CLI / Arduino IDE — `.ino` sketches under `firmware/{atom-lite,atoms3-lite}/`
- C++17 with ArduinoJson, ESPAsyncWebServer (`AsyncAuthenticationMiddleware` from 0.1.9)
- Vanilla HTML/CSS/JS (no React, no external CDN — enforced by `tests/web_ui_index.test.mjs`)
- Node `node:test` for tests (under `tests/`), `node:vm` for inline JS unit tests
- `tools/sync_shared.sh` for 3-way mirror sync, `tools/embed_web_ui.mjs` for gzip embed

## Spec Deltas (corrections applied during planning)

The committed spec (`docs/superpowers/specs/2026-05-29-web-admin-header-redesign-design.md`) was authored before the existing JSON envelope and field convention were re-inspected. The following corrections apply to **this plan and the implementation**; the spec text is not amended (operates as the design north-star).

| Spec text | Actual implementation | Reason |
|---|---|---|
| `metrics.freeHeap` (new field) | use existing `metrics.heap_free` | Already present at `device_status.cpp:105` since 0.1.1 |
| `metrics.authFailures` | `metrics.auth_failures` | Existing JSON uses snake_case throughout |
| `metrics.lastAuthFailMs` | `metrics.last_auth_fail_ago_ms` (computed at serialize) | Matches `last_rx_ago_ms` pattern at `device_status.cpp:120` |
| `state.wifi.*` / `state.metrics.*` (JS) | `state.data.wifi.*` / `state.data.metrics.*` | Existing envelope is `{ok:true, data:{...}}` |
| `/api/info` flat object | `{ok:true, data:{firmware:{...}, tcp:{...}, queue:{...}}}` | Match the project's envelope convention |
| `BUILD_COMMIT` / `BUILD_AT` via PlatformIO | Generated into `shared/src/build_info.h` by `tools/sync_shared.sh` | Project uses Arduino-CLI, not PlatformIO |
| Spec D2 "alert count + reboot" / DOM `id="navAlertCount"` | DOM ids prefixed `hdr` (`hdrHealth`, `hdrAlertCount`, `hdrReboot`) | Avoid collision with existing `navIp`/`navRssi`/`navUptime` ids; the old top-state span will be removed by Task 15 |
| Drawer JS reads `state.data.tcp.max_clients` for TCP "n/n" display | Same field already exposed at `device_status.cpp:85` | No new server field for `max_clients` needed |

`queue.capacity` is genuinely new and must be added to `/api/info` (Task 7).

<script id="header-pure-fns">
function wrapDelta(curr, prev, max) {
  max = max || 0xFFFFFFFF;
  return curr >= prev ? (curr - prev) : ((max - prev) + curr + 1);
}

function deriveRates(prev, curr, dtSec) {
  if (!prev || dtSec <= 0) return { rxPktRate: 0, rxByteRate: 0, txPktRate: 0, txByteRate: 0 };
  return {
    rxPktRate:  Math.round(wrapDelta(curr.rxPackets, prev.rxPackets) / dtSec),
    rxByteRate: Math.round(wrapDelta(curr.rxBytes,   prev.rxBytes)   / dtSec),
    txPktRate:  Math.round(wrapDelta(curr.txPackets, prev.txPackets) / dtSec),
    txByteRate: Math.round(wrapDelta(curr.txBytes,   prev.txBytes)   / dtSec),
  };
}

function computeHealth(state, activeAlerts) {
  if (!state) return "offline";
  if (activeAlerts && activeAlerts.some(function (a) { return a.severity === "critical"; })) return "fail";
  if (activeAlerts && activeAlerts.some(function (a) { return a.severity === "warning";  })) return "warn";
  return "ok";
}

function formatUptime(secs) {
  secs = Math.max(0, Math.floor(secs || 0));
  var d = Math.floor(secs / 86400);
  var h = Math.floor((secs % 86400) / 3600);
  var m = Math.floor((secs % 3600) / 60);
  var s = secs % 60;
  if (d > 0) return d + "d " + h + "h " + m + "m";
  if (h > 0) return h + "h " + m + "m " + s + "s";
  if (m > 0) return m + "m " + s + "s";
  return s + "s";
}

if (typeof window !== "undefined") {
  window.__headerExports = { wrapDelta, deriveRates, computeHealth, formatUptime };
}
